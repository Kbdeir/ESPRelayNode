Node Configuration
1. Physical location tag: Node loaction e.g. Ceiling light at main bedroom.
MQTT configuration
* Pic Interface: set to 1 if board is interface to KAYB 1
* MQTT publishing: select to enable MQTT protocol communication

* MQTT server address: IP address of MQTT broker
* MQTT Port: Port number of MQTT broker

*Default MQTT Publish Topic: the MQTT Topic to which this board will publish its updates and commands

*MQTT Subscribtion Topic MQTT topic to subscribe to.
Switching configuration
1. Restore Coil Status on Restart : Tick to restore last coil status on restart

2.Time to Live (TTL) in seconds: Once turned ON, the relay will turn OFF automatically after this number of seconds. The board will publish a MQTT message on status change.

3.Time to Activate (TTA) in seconds: Delay in seconds before the relay is activated.

4. Maximum current allowed before coil shutdown: Maximum user defined current allowed through the relay. Above this threshold, the relay will shut itshelf down for 30 seconds. During this time the board is NOT responsive to Modbus or MQTT commands. Normal operation will be resumed after 30 seconds. The board will publish a MQTT message on status change.

5. Unit Max Current: maximum current the Power sensor can measure. should be 20 A or 30 A
