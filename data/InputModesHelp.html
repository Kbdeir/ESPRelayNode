Quick Choice

MQTT pulse

Click Mode

Use this for a push button that should publish a single toggle command.

When the input is pressed, the device publishes tog to the input MQTT topic.

MQTT state

Input Mode

Use this when you want to monitor the real input state.

Every change publishes on or off to the input MQTT topic.

Relay toggle

Toggle Mode - Coil Connected

Use this for a push button that directly toggles a relay.

Press once to switch the mapped relay. If the relay has a delayed-on time, that timer is used when turning on.

Relay follows input

Copy Input to Coil

Use this for a switch, float sensor, contact, or controller output where the relay should follow the input.

Input high turns the mapped relay on. Input low turns it off.

Relay Mapping

Click Mode and Input Mode only publish MQTT messages by themselves.

Toggle Mode - Coil Connected and Copy Input to Coil need a mapping in the Input to Relay Map page. The map tells the firmware which input controls which relay.

  • Open Input to Relay Map.
  • Select the physical input on the left.
  • Select the relay number on the right.
  • Save, then restart if you need the boot-time wiring to reload cleanly.
Input 0 on GPIO4 is also the configuration button on this ESP32 build. The firmware mainly wires Input 1 through Input 6 as normal physical inputs.

Temperature Sensor

Some inputs also show Temperature Sensor. Use that only for DS18B20 temperature sensor pins. In that mode the input is not watched as a button or relay control input.
