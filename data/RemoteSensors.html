Remote Sensor Sources

Map MQTT topics published by other boards to numbered sensor slots. Slots appear as condition sources in Automation Rules.

How it works

The remote board publishes its sensor value to an MQTT topic. This board subscribes and caches the latest value. Set JSON Key if the payload is a JSON object (e.g. {"realPower":2450} → key = realPower); leave blank if the payload is a plain number. Each filled slot appears as Remote N: Label in Automation Rule condition dropdowns.
