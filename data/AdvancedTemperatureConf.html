🔌 Wiring & Pin Assignment — DS18B20 Temperature Sensors

Two DS18B20 OneWire sensors are required. Each sensor uses its own dedicated GPIO pin (separate OneWire buses — they are not sharing a single bus).

  Both sensors may share the same +3.3V/+5V and GND rails,
  but each DATA line needs its own 4.7k ohm pull-up resistor.

Pull-up resistor (required on each DATA line)

Place a 4.7 kΩ resistor between DATA and VCC for each sensor. Without it the OneWire bus will not communicate reliably. If you power the sensor from 3.3 V, pull up to 3.3 V; if from 5 V, pull up to 5 V (GPIO33 and GPIO16 are both 5 V-tolerant input pins on this board).

Parasitic power (2-wire) vs. normal power (3-wire)

In parasitic mode VCC is left unconnected and the sensor draws power from the DATA line — only GND and DATA are wired. This works but is less reliable over long cable runs or at low temperatures. Normal 3-wire mode (VCC + GND + DATA) is recommended.

Input pin conflict

While temperature control is active, GPIO33 (Input 1) and GPIO16 (Input 2) are owned by the OneWire drivers and cannot be used simultaneously for relay-toggle inputs or other input functions.

Two temperature sensors are required: Sensor 1 (GPIO33) measures the solar panel temperature, and Sensor 2 (GPIO16) measures the water tank temperature. Control runs only when Enable temperature control is checked.

Relay ON condition — all of the following must be true
  • Enable temperature control is checked.
  • Low Temp ≤ Panel temp ≤ High Temp — panel is within the configured operating range.
  • (Panel temp − Tank temp) > Buffer — panel is significantly hotter than the tank.
Relay OFF condition — any of the following
  • Enable temperature control is unchecked.
  • Panel temp < Low Temp — panel is not warm enough.
  • Panel temp > High Temp — panel is overheating (only checked when High Temp ≠ 0).
  • (Panel temp − Tank temp) < Buffer — panel and tank temperatures are too close.
Example

Low Temp = 40°C, High Temp = 90°C, Buffer = 8°C:
• Panel 55°C, Tank 44°C → diff = 11°C > 8 → Relay ON
• Panel 55°C, Tank 50°C → diff = 5°C < 8 → Relay OFF
• Panel 35°C, Tank 30°C → panel < 40°C → Relay OFF
• Panel 95°C, Tank 40°C → panel > 90°C → Relay OFF

Notes
  • If High Temp is set to 0, the upper bound check is skipped.
  • Control is skipped entirely if Low Temp or Buffer is set to 0.
